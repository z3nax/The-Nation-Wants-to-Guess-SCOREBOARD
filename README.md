# The-Nation-Wants-to-Guess-SCOREBOARD

🎮 How It Works
1. Main Scoreboard Arena
Change generic labels ("Panelist 1", etc.) by typing directly into the input field names. Changes reflect automatically across the entire app.

Use the green and red buttons to quickly control points during fast-paced play.

Use the Target Add box to type a specific custom value and hit 🎯 Add.

2. "Who Am I?" Final Round Dashboard
Each panelist's updated bank value is synced onto their final card.

Type their desired risk profile value into the Wager 💰 text field.

Click 🎉 Win to credit the wager amount to their core score, or click 💥 Lose to deduct it instantly.
